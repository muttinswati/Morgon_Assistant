# Morgan – Voice Assistant

**Morgan** is a Python-based voice assistant that executes tasks via voice commands: web search, YouTube playback, Wikipedia queries, emails, system control, and launching apps.  

## Key Features
- Voice-controlled interaction and feedback  
- Wikipedia summaries and YouTube playback  
- Web browsing (Google, YouTube, Brave)  
- Email automation via Gmail  
- System operations: shutdown, restart, sleep  
- Launch local apps like VS Code and Notepad  

## Technologies
- **Python 3**, `pyttsx3`, `speech_recognition`, `wikipedia`, `pywhatkit`, `webbrowser`, `smtplib`, `os`  

## How It Works
1. Greets user based on time.  
2. Listens and recognizes voice commands.  
3. Performs tasks based on keywords.  
4. Provides voice feedback.  
5. Runs continuously until stopped.
